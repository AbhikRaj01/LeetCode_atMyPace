# THE RULES

## Rule 01: There is no motivation.

Motivation is temporary.

Discipline is permanent.

Code when you are excited.

Code when you are tired.

Code when your brain tells you:

> "Skip today. It won't matter."

That voice has destroyed more dreams than failure ever will.

---

## Rule 02: Stop negotiating with yourself.

The moment you start saying:

> "I'll start tomorrow."

You already lost today's battle.

The successful version of you is built from the days you did not want to work.

---

## Rule 03: Be willing to look stupid.

Staring at a problem for three hours and understanding nothing?

Good.

That is your brain fighting a war against its old limitations.

The comfortable mind does not evolve.
